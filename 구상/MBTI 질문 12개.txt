[ E/I, 외향형/내향형 질문]
Q. 오늘은 드디어 첫 녹화날! 스튜디오에서의 나는
1. 하 너무 떨린다…. 잘 할수 있을까…? 말도 잘 못꺼낼거같은데… ( I )
2. 첫 녹화 당당하게 도전~. 난 자신있어~ 사람들 앞에서 입좀 털어볼까?( E )

Q. 오늘은 간만에 스케쥴이 없는날! 나는
1. 편안한 내 공간에서 넷플릭스 정주행하면서 집콕방콕 뒹굴뒹굴 ( I )
2. 무적권 약속 잡은후에 집밖으로 탈출해서 지인들만나기 ( E )

Q. 연인과 길을 걷다 연인의 동창을 만났다! 이때의 나는
1. 마치 알던 사이인 것처럼 같이 이야기를 나눈다.( E )
2. 난 투명인간이야... 땅만 보고 대화가 끝나기를 기다린다. ( I )

[ N/S, 감각형/직관형 질문 ]
Q. 오늘은 몸관리를 위해 운동을 하는 날! 운동을 끝낸 후 드는 생각은?
1. “오늘 5km나 뛰었네. 내일도 이만큼 도전!”이라고 생각하며 바로 씻으러 간다. ( S )
2. 하 이정도 땀이면 지방도 빠지고 근육도 좀 생긴거같은데? 복근이 생긴 것 같은 느낌적인 느낌~? ( N )

Q. 같이 데뷔한 동료가 자신의 힘든 상황에 대해 이야기한다. 이때의 나는?
1. 상황을 해결하기 위한 현실적인 해결책을 제시해준다. ( N )
2. 상대의 감정에 공감하고 정신적인 지지를 해준다. ( S )

Q. 배가고파서 차타고 맛집을 찾고있는 당신! 이 때 매니저가 길을 모른다고 하는데, 당신은?
1. 쭉 내려가서 왼쪽으로 돌면 돼요 ! ( N )
2. 100m 직진해서 편의점 끼고 우회전 해주세요! ( S )

[ T/F , 사고형/감정형 질문 ]
Q. 비밀연애를 하고 있는 나! 남자친구/여자친구가 3시간 넘게 연락이 없을 때 나는?
1. 손가락이 부러졌나? 아니다 발가락인가? 수시로 핸드폰만 들여다본다.(f)
2. 일이 바쁜가 봉가~ 하면서 별로 신경 안쓴다.(t)

Q. 하,, 이제 곧 내 생일인데? 내 주변 지인들이 내 생일선물 챙겨줄때
1. 물어보고 나한테 필요한걸 줬으면 좋겠다. ( T )
2. 마음이 느껴지기만 하면 어떤 선물이라도 OK ( F )

Q. 친구가 우울해서 화분을 샀다고 한다. 그때의 나는?
1. 왜 화분을 사..? ( T )
2. 헐 왜 우울해!! 무슨 일 있어..?? ( F )

[ J/P , 판단형/인식형 질문 ]
Q. 예능 프로그램에서 놀이동산에 온 당신 ! 놀이기구를 탈 때의 나는?
1. 오늘 이거 다섯 개는 꼭 타야돼! 지도를 보며 계획을 세운다. ( J )
2. 오, 저거 재밌어 보인다! 일단 보이는 것부터 탄다. ( P )

Q. 이제 곧 동료의 생일! 동료의 선물을 준비할 때 나는
1. 계획이 다 있구나..미리 생각해서 사놓는다. ( J )
2. 헉! 어떡해! 생일 직전에 급박하게 산다. ( P )

Q. 동료들과 함께 앞으로의 신년계획과 관련된 회의를 했다! 긴 회의를 마무리할 때의 나는
1. 오늘 정한대로 하나하나 진행하자! ( J )
2. 진행되는거 봐가면서 하나하나 계획을 정하자 ( P )
